# Google API

現在許多網站平台開放物件功能與資訊，提供其它網站進行串接及共享，從中達到平台的市佔性。
這些平台開放分享的資訊，進行打包成可物件連結的 API 介面，提供其它應用程式串接，就是 API 的核心！
API 能取得大眾上傳的資訊，也能藉由這些已知的資訊提供服務給大眾，算得上是互利共生。

最常見的 API 是 FB 和 Google 的登入系統，方便第三方進行功能擴充，對第三方來說，省時省力。
